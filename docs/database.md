# Документация по таблицам базы данных

| Поле             | Тип             | Обязательное | Описание                                        |
| ---------------- | --------------- | -----------: | ----------------------------------------------- |
| `id`             | `uuid`          |           Да | Уникальный идентификатор пользователя (uuid v7) |
| `name`           | `varchar(255)`  |           Да | Полное имя пользователя                         |
| `last_name`      | `varchar(255)`  |           Да | Фамилия пользователя                            |
| `first_name`     | `varchar(255)`  |           Да | Имя пользователя                                |
| `email`          | `varchar(255)`  |           Да | Email пользователя (уникальный)                 |
| `email_verified` | `boolean`       |           Да | Подтвержден ли email                            |
| `image`          | `varchar(1024)` |          Нет | URL изображения (аватар)                        |
| `tz`             | `varchar(50)`   |           Да | Таймзона пользователя                           |
| `created_at`     | `timestamptz`   |           Да | Дата создания                                   |
| `updated_at`     | `timestamptz`   |           Да | Дата обновления                                 |

**Назначение:** хранение пользователей системы.
