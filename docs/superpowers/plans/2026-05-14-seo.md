# SEO Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Внедрить production-уровень SEO для публичных страниц `apps/web` (главная, `/pricing`, `/terms`, `/terms/[document]`) с прицелом на Яндекс + Google.

**Architecture:** Изолированный модуль `apps/web/src/lib/seo/` содержит фабрику `buildMetadata`, компонент `<JsonLd>` и 6 Schema.org-фабрик. Корневой `layout.tsx` задаёт `metadataBase` и верификации. Динамические OG-картинки через Next.js file-conventions (`opengraph-image.tsx`). `robots.ts` + `sitemap.ts` собираются динамически из `legalDocuments`. Расширение под будущие marketing-страницы — добавление одной строки `export const metadata = buildMetadata(...)`.

**Tech Stack:** Next.js 16 (App Router), TypeScript, React 19, vitest (unit), Playwright (E2E), `next/og` (edge runtime для OG-картинок).

**Spec reference:** [docs/superpowers/specs/2026-05-14-seo-design.md](../specs/2026-05-14-seo-design.md)

```ts
import type { Metadata } from 'next'

import { siteConfig } from './site-config'

export type BuildMetadataInput = {
  title: string
  description?: string
  path: string
  ogImage?: string
  noIndex?: boolean
  keywords?: string[]
}

export function buildMetadata(input: BuildMetadataInput): Metadata {
  const url = new URL(input.path, `${siteConfig.url}/`).toString()
  const description = input.description ?? siteConfig.description
  const robots = input.noIndex
    ? { index: false, follow: false }
    : { index: true, follow: true, googleBot: { index: true, follow: true } }

  return {
    title: input.title,
    description,
    keywords: input.keywords,
    alternates: { canonical: url },
    robots,
    openGraph: {
      type: 'website',
      url,
      title: input.title,
      description,
      siteName: siteConfig.name,
      locale: siteConfig.locale,
      images: input.ogImage ? [input.ogImage] : undefined,
    },
  }
}
```

```tsx
import 'server-only'

type JsonLdData = Record<string, unknown> | Record<string, unknown>[]

type JsonLdProps = { data: JsonLdData | null }

export function JsonLd({ data }: Readonly<JsonLdProps>) {
  if (data === null) return null
  const json = JSON.stringify(data).replace(/</g, '\\u003c')
  return <script type="application/ld+json" dangerouslySetInnerHTML={{ __html: json }} />
```
