# Onboarding

Процесс онбординга новых пользователей AnyNote.

## Шаги регистрации

1. Пользователь вводит email, имя, фамилию и пароль.
2. Система отправляет письмо для подтверждения email через SendSay.
3. После подтверждения пользователь попадает на страницу принятия соглашений.
4. Пользователь принимает USER_AGREEMENT, PRIVACY_POLICY, PII_PROCESSING, PUBLIC_OFFER.
5. После принятия соглашений создаётся первое рабочее пространство.

## Технические детали

Регистрация реализована через better-auth с полями firstName и lastName.

Email-верификация обязательна. Без неё пользователь не может использовать приложение.

Google OAuth также поддерживается через socialProviders.google.
