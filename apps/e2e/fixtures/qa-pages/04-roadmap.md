# Roadmap Q3

## Запланированные функции

### Q3 2026

- Agent OS v1: LangGraph-агент с памятью, MCP-клиентом и подтверждением деструктивных операций.
- Улучшенный поиск: векторный + полнотекстовый поиск по всем страницам рабочего пространства.
- Коллаборация: улучшенный YJS-сервер с оптимистичными обновлениями.

### Q4 2026 (предварительно)

- Страничные агенты (page agents): автономные агенты, привязанные к конкретным страницам.
- Bulk-подтверждения: подтверждение нескольких деструктивных операций одним кликом.
- Observability: OpenTelemetry трассировка для агентских запросов.

## Деferred (вне текущего scope)

- OAuth MCP-серверы
- Выбор модели по ролям
- LLM-compaction памяти
